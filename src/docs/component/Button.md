# Button 按钮

常用的操作按钮

## 基础用法
基础的按钮用法

<ClientOnly>
 <button-demo-base></button-demo-base>
</ClientOnly>

::: tip

使用`type`、`plain`、`round`来定义 Button 的样式

:::

```vue
  <ml-button>默认按钮</ml-button>
  <ml-button type="primary">主要按钮</ml-button>
  <ml-button type="success">成功按钮</ml-button>
  <ml-button type="primary">主要按钮</ml-button>
  <ml-button type="success">成功按钮</ml-button>
  <ml-button type="primary">主要按钮</ml-button>
```
